# lagermanager
Einfaches Bestell und Lager Tool

#Docker-Web-App
Installieren Sie Docker
führen Sie im Hauptordner (wo die docker-compose.yml Datei liegt) den folgenden Befehl aus: docker compose up -d --build

Die Anwendung wird anschließend ausgerollt und ist erreichbar unter:
http://localhost:3000

Daten werden in einer CSV-Datei gespeichert/verwaltet.
Funktionen:
  - Desktop Ansicht und Mobile Ansicht
  - Unterteilung: Offen, Teillieferung, Abgeschlossen
  - Import / Export
  - Bestellungen als Projekt definieren
  - Lieferscheine / Dokumente verknüpfen
  - Als ausgeliefert markieren
  - Volltextsuche
  - Abgeschlossene Einträge archivieren
  - Archivverwaltung mit separater Suchfunktion
